Instrucciones:
1. Subir carpeta 'fernandccessible' a /wp-content/plugins/.
2. Asegúrate de que 'assets/accessibility.min.js' contiene tu script (reemplaza el de ejemplo si es necesario).
3. Activar el plugin en el panel de WordPress.
4. El menú se inyectará automáticamente en el footer. También puedes insertar manualmente el contenedor con el shortcode: [fernand_accessibility_menu]
