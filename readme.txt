Instrucciones:
1. Subir carpeta 'fernandccessible' a /wp-content/plugins/.
2. Asegurate de que 'assets/accessibility.min.js' contiene tu script (reemplazá el de ejemplo si es necesario).
3. Activar el plugin en el panel de WordPress.
4. El menú se inyectará automáticamente en el footer.
